### 🏎️ E-Baja — Brakes Subsystem

Working on the design, analysis, and testing of the braking system for our college's E-Baja vehicle — covering brake force calculations, caliper/disc selection, hydraulic circuit design, and on-track validation.

- Brake system design & calculations
- CAD modeling (SolidWorks / Fusion 360)
- FEA / simulation validation
- Testing & data logging

I like combining mechanical engineering concepts with web development — building calculators, visualizers, and simulators for topics I study.

| Project | Description | Tech | Link |
